# Pizzaria DB - Projeto de Banco de Dados

## Descrição

Este projeto tem como objetivo a criação de um banco de dados para gerenciar informações de uma pizzaria, incluindo o armazenamento de pedidos, clientes, produtos, entre outros. Ele foi desenvolvido como parte de um estudo sobre arquitetura de dados e otimização de consultas em sistemas de banco de dados.

## Funcionalidades

- Cadastro de clientes
- Gestão de pedidos
- Controle de estoque de produtos
- Relatórios de vendas e métricas de desempenho
